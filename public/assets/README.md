# Game Assets

Drop assets here. Phaser loads from `/assets/...`.

## Required images (`images/`)

- `background.png` — 800×1600 stadium at night
- `goal.png` — 600×350 goal post, transparent bg
- `ball.png` — 64×64
- `crowd.png` — 800×350 crowd silhouette
- `net_flash.png` — 600×300 white mesh w/ alpha
- `keeper.png` — 512×196 (4-frame spritesheet, 128×196 per frame)

## Required audio (`audio/`)

- `crowd_ambient.mp3`
- `kick.mp3`
- `goal.mp3`
- `save.mp3`
- `miss.mp3`
- `whistle.mp3`

## Free sources

- Ball: https://opengameart.org/content/soccer-ball
- Keeper: https://kenney.nl/assets/sports-pack
- SFX: https://freesound.org (search "crowd cheer", "football kick", "whistle")
